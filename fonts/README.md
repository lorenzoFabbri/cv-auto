# Vendored fonts

These fonts are committed so the Typst PDF renders identically on every machine
(local macOS, CI, any clone) with no missing-font warnings. They are resolved via
the `font-paths: [fonts]` setting in `_quarto.yml`.

| File(s) | Family | Used for | License |
|---|---|---|---|
| `LiberationSans-{Regular,Bold,Italic,BoldItalic}.ttf` | Liberation Sans | All body text | SIL Open Font License 1.1 (see `LICENSE`) |
| `fa-solid-900.ttf` | Font Awesome 6 Free (Solid) | Header icons: email, location, website, Google Scholar | SIL OFL 1.1 (Font Awesome Free) |
| `fa-brands-400.ttf` | Font Awesome 6 Brands | Header icons: ORCID, GitHub, LinkedIn, Bluesky | SIL OFL 1.1 (Font Awesome Free) |

Font Awesome Free — https://fontawesome.com — fonts under SIL OFL 1.1, icons under
CC BY 4.0. Version 6.7.2 (sourced from the Quarto fontawesome extension).
